Now that you've set up your first Cube deployment, learn more about what else
Cube Cloud can do for you.

## Access Control

Cube Cloud enhances the column and row-level security available in open-source
and adds controls for securing access to the Cube Cloud UI and individual
deployments.

## Alerts

You can send alerts generated by Cube Cloud to users or custom email addresses,

## Observability

The Queries and Metrics pages in Cube Cloud provide query-level observability,
allowing for easy diagnosis of performance issues and potential improvements.
